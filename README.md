# ARQA · أرقى

**A**rchitectural · **R**endering · **Q**uantification · **A**utomation

A multi-agent LLM system for end-to-end architectural design — generating floor plans, technical reports, and bilingual documentation from natural language briefs in English, Urdu, and Arabic.

## Phase 1 (Days 1–20)

Building the foundation:

- **Supervisor Agent** — orchestration, blackboard state, quality validation, revision loop
- **Communication Agent** — multilingual NLP (EN / UR / AR), structured brief extraction
- **Design Generation Agent** — floor plan layouts, DXF output, cultural compliance
- **Report Writing Agent** — trilingual PDFs, material takeoff, multi-currency costing (PKR / SAR / AED / QAR)
